COPYCAT

Support

COPYCAT runs inside ChatGPT. It does not require a COPYCAT account, API key, browser bridge, local server, or separate login.

How to play

  1. Install or enable COPYCAT from ChatGPT's Plugins Directory.
  2. Start a new chat and ask, “Let’s play COPYCAT.”
  3. Enter a nickname, then choose online matchmaking, quick CPU practice, ChatGPT CPUs, or a private room.
  4. Give a one-word hint, vote for the Copycat, and complete the final guess if the Copycat is caught.

If COPYCAT is still in review and not yet listed publicly, developers and reviewers can add the production MCP endpoint in ChatGPT Developer Mode using https://copycat-chatgpt-app.kistame228.workers.dev/mcp with no authentication.

Common fixes

  • The widget does not open: start a new chat, confirm the COPYCAT plugin is enabled, and ask explicitly to open COPYCAT.
  • No players are available: online matchmaking automatically adds rule CPUs after about ten seconds, or you can choose Quick CPU practice immediately.
  • A room code fails: confirm all four characters and ask the host to keep the room open. Expired rooms cannot be restored.
  • The game disconnected: keep the chat open and retry. The widget automatically attempts to reconnect.

Report a problem

Open a GitHub issue with what you expected, what happened, your device and ChatGPT surface, and the approximate time. A room code may help diagnose a live-room problem, but never include passwords, API keys, full chat transcripts, or sensitive personal information.
